# HTML

### HTML(Hypertext markup language) 超文本标记语言

#### 背景知识

1. 2014年 W3C推``HTML5``

3. ``HTML``文件：
   1. 不需要编译，浏览器解释执行
   2. 通常后缀``.html``，``.htm``
   3. 大小写不敏感的文本文件

#### HTML基本结构

```html
<html>
 	<head>
    	<title>标题</title>
  </head> 
 	<body>
    	网页主体
  </body>
</html>
```

#### HTML 基础和细节
1. ``html``文件由一个个嵌套的``元素``组成，即``<标签名 属性1=“属性值” 属性2=“属性值2”...> 标签内容 <\标签名>``

2. ``注释``  =  <!-- 注释这么写-->

3. 声明HTML``版本类型``，需用刀<!DOCTYPE html>，该标签需要放置到文档的首行。

4. 有时候，尤其是在windows平台下，打开编写的html会出现中文乱码的情况。
