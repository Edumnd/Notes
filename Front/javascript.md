# JavaScript

JavaScript 是一种基于``对象``和``事件驱动``的客户端脚本语言



## Javascript 基础

#### 注释

```javascript
//  javascript 用 // 注释单行
/*  javascript 用该方法注释多行  */
//  javascript 语句用  ;  结束
```



#### 标识符

变量、函数、属性名称：

- 字母、数字、下划线_、美元符号$
- 不能数字开头
- 不能使用关键字



#### 变量定义和赋值

- 用``var``声明变量、变量赋值

  ```javascript
  var student_name;    // 声明变量
  var student_$_ID;    // 都合法，只是不能数字开头
  var student_hobby="football";
  var gender;
  gender="male";
  var age=18, email="ssx@gmail.com", address;  //声明和赋值多个变量
  
  family="no family";    //省略var，则定义全局变量。该方式不推荐。
  ```
